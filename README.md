# Simple Static Site (Ready for GitHub Pages)

This is a minimal static HTML/CSS/JS project you can host on **GitHub Pages**.
Folder structure is ready to push to a repository and enable Pages from `main` or `gh-pages` branch.

## How to use
1. Create a new GitHub repository.
2. Copy the files from this folder into the repo (or push this folder).
3. In repository settings, enable GitHub Pages and choose the `main` branch (root) or `gh-pages` branch.
4. Visit `https://<your-username>.github.io/<repo-name>/` after Pages builds.

## Files
- `index.html`, `about.html`, `contact.html` — pages
- `css/styles.css` — styles
- `js/script.js` — small client logic
- `images/logo.svg` — logo
